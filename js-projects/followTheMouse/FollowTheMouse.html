On this page, create an effect where an image follows the mouse around (you choose the image but keep it kind of small). Make it so that the image can be in front of or behind the text.

  • Look into adding an event listener for mouse movements. Add this to the whole document (so document.addEventListener...)
  • Look into getting the mouse's x and y coordinate in the callback method that you setup for mouse movements.
  • You will need to modify the CSS related to the image...to modify CSS in JavaScript, you will modify the style property and then further subproperties of the style.
    • You probably want to set the style's "position" property
    • You also want to set the top and left property
    • You will want to set the position of the image to be a couple of pixels to the right and below the mouse's actual coordinates. If you don't do this, then all mouse clicks will be on the image, which will make it impossible to click on the buttons for the challenge.
  • For the last part, putting the image in front of or behind the text, look into the zIndex property (a CSS property).
